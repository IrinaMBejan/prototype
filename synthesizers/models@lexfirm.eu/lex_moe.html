About the Synthesizer

Lex-MoE is a specialized mixture-of-experts model designed for European legal reasoning and analysis. It combines multiple expert models to handle different aspects of legal work, from document analysis to citation generation.

Model Architecture

  • Mixture-of-Experts (MoE) architecture with specialized legal experts
  • Expert routing based on query type and legal domain
  • Multilingual support for EU languages (EN, DE, FR, IT, ES)
  • Optimized for legal document understanding and reasoning

Expert Models

The model includes specialized experts for extractive question answering, citation generation, legal argumentation, and multilingual statute interpretation. Each expert is fine-tuned on specific legal tasks and domains.

Training Data

Trained on comprehensive EU legal corpus including treaties, directives, regulations, case law from European Court of Justice, and national legal systems. The model has been validated against established legal databases and practitioner feedback.

Access the Synthesizer

Python SDK

Use the SyftBox Python client to interact with the synthesizer:

#!/usr/bin/env python3
"""
Lex-MoE synthesizer example
"""
import syft_hub as sh

def main():
    # Initialize the SyftBox client
    client = sh.get_client()
    
    # Connect to the Lex-MoE synthesizer
    synthesizer = client.get_service(
        datasite="lexfirm.eu",
        name="lex_moe"
    )
    
    # Send a legal query
    response = synthesizer.chat(
        message="What are the key requirements for GDPR compliance in data processing?",
        temperature=0.3
    )
    
    # Display response
    print("Lex-MoE Response:")
    print("-" * 40)
    print(response.get('content', ''))
    print()
    print(f"Tokens used: {response.get('usage', {}).get('total_tokens', 'N/A')}")

if __name__ == "__main__":
    main()
