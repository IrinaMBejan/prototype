About the Synthesizer

BioBERT Clinical is a state-of-the-art biomedical language model that has been pre-trained on a large corpus of biomedical literature and clinical notes. It's specifically designed to handle the complex terminology and structures found in medical and scientific texts.

Model Architecture

  • Based on BERT (Bidirectional Encoder Representations from Transformers)
  • Pre-trained on PubMed abstracts and clinical notes
  • Fine-tuned for biomedical NLP tasks
  • Optimized for medical entity recognition and extraction

Capabilities

The model excels at understanding medical terminology, extracting clinical entities, and processing biomedical text with high accuracy. It can handle complex medical queries and provide detailed responses based on the latest research.

Training Data

Trained on a comprehensive corpus including PubMed abstracts, clinical notes, medical textbooks, and research papers. The model has been validated against established medical databases and clinical guidelines.

Access the Synthesizer

Python SDK

Use the SyftBox Python client to interact with the synthesizer:

#!/usr/bin/env python3
"""
BioBERT Clinical synthesizer example
"""
import syft_hub as sh

def main():
    # Initialize the SyftBox client
    client = sh.get_client()
    
    # Connect to the BioBERT Clinical synthesizer
    synthesizer = client.get_service(
        datasite="mayo.edu",
        name="biobert_clinical"
    )
    
    # Send a medical query
    response = synthesizer.chat(
        message="What are the common side effects of statins?",
        temperature=0.7
    )
    
    # Display response
    print("Medical Response:")
    print("-" * 40)
    print(response.get('content', ''))
    print()
    print(f"Tokens used: {response.get('usage', {}).get('total_tokens', 'N/A')}")

if __name__ == "__main__":
    main()

HTTP API

Make direct HTTP requests to the synthesizer API:

# Chat API Example
curl -X POST "https://api.syftbox.net/v1/services/biobert-clinical/chat" \\
  -H "Content-Type: application/json" \\
  -H "Authorization: Bearer YOUR_API_KEY" \\
  -d '{
    "message": "What are the common side effects of statins?",
    "temperature": 0.7,
    "max_tokens": 1000,
    "stream": false
  }'
