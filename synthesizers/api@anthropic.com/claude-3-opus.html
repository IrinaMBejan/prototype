Access the Synthesizer

Python SDK

Use the SyftBox Python client to interact with the synthesizer:

#!/usr/bin/env python3
"""
Claude 3 Opus synthesizer example
"""
import syft_hub as sh

def main():
    # Initialize the SyftBox client
    client = sh.get_client()
    
    # Connect to the Claude 3 Opus synthesizer
    synthesizer = client.get_service(
        datasite="anthropic.com",
        name="claude-3-opus"
    )
    
    # Send a complex reasoning query
    response = synthesizer.chat(
        message="Solve this math problem step by step: A train leaves station A at 2 PM traveling at 60 mph. Another train leaves station B at 3 PM traveling at 80 mph. If the stations are 300 miles apart, when will they meet?",
        temperature=0.3
    )
    
    # Display response
    print("Claude Response:")
    print("-" * 40)
    print(response.get('content', ''))
    print()
    print(f"Tokens used: {response.get('usage', {}).get('total_tokens', 'N/A')}")

if __name__ == "__main__":
    main()

Model Context Protocol (MCP)

Installation Options

Claude MCP CLI (Recommended)

Install using the Claude MCP command line interface:

claude mcp add --transport http syftbox_claude3_opus https://mcp.syftbox.net/claude-3-opus

Usage

Once installed, the Claude 3 Opus service will be available as an MCP tool in Claude Desktop conversations.

Example usage in Claude:

"Use the claude3_opus tool to solve complex reasoning problems..."

Verification

After installation, you should see "Claude 3 Opus" listed in your Claude Desktop MCP tools list.

Manual Installation

Step 1: Install the MCP server package
npm install -g @syftbox/mcp-claude3-opus
Step 2: Add to Claude Desktop config

Edit your Claude Desktop configuration file:

macOS/Linux: ~/.config/claude/claude_desktop_config.json
Windows: %APPDATA%\\Claude\\claude_desktop_config.json

Add this configuration:

{
  "mcpServers": {
    "claude3_opus": {
      "command": "mcp-claude3-opus",
      "args": [
        "--api-key", "YOUR_API_KEY",
        "--endpoint", "https://api.syftbox.net"
      ],
      "env": {
        "SYFTBOX_SERVICE_ID": "claude-3-opus",
        "SYFTBOX_DATASITE": "anthropic.com"
      }
    }
  }
}
Step 3: Restart Claude Desktop

After saving the configuration, restart Claude Desktop for the changes to take effect.
