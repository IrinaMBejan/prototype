Operational records, alerts, and manufacturer bulletins from hospital device management systems. PHI masked at source via SyftBox policies. This dataset includes device logs, maintenance schedules, safety alerts, UDI registry links, and operational status reports from critical medical equipment across multiple hospital departments.

About

This dataset contains anonymized operational data from medical device management systems across multiple hospital departments, with all PHI removed at source.

Data Sources

  • Hospital device management systems
  • Manufacturer safety bulletins and recalls
  • Maintenance logs and service records
  • Device performance monitoring systems

Coverage

The dataset covers 12 hospital departments including ICU, Emergency, Surgery, and Radiology. Device types include ventilators, monitors, infusion pumps, and imaging equipment.

Privacy & Compliance

All PHI is masked at source using SyftBox privacy policies. The dataset maintains full HIPAA and GDPR compliance while preserving operational insights for research and safety analysis.

Python SDK
Python 
from syftbox import Dataset

# Load the dataset
devices = Dataset('admin@st-marys-hospital.com/meddevice_records')

# Basic search
results = devices.search('ventilator maintenance', top_k=10)

# Advanced search with filters
results = devices.search(
    'critical alerts ICU',
    top_k=20,
    filters={

        'alert_level': 'Critical',
        'status': 'Alert'
    }
)

Model Context Protocol (MCP)

Installation Options

Claude MCP CLI (Recommended)

Install using the Claude MCP command line interface:

claude mcp add --transport http syftbox_meddevices https://mcp.syftbox.net/med-devices

Usage

Once installed, the MedDevice Records service will be available as an MCP tool in Claude Desktop conversations.

Example usage in Claude:

"Use the meddevices tool to search for information about..."

Verification

After installation, you should see "MedDevice Records" listed in your Claude Desktop MCP tools list.

Manual Installation

Step 1: Install the MCP server package
npm install -g @syftbox/mcp-meddevices
Step 2: Add to Claude Desktop config

Edit your Claude Desktop configuration file:

macOS/Linux: ~/.config/claude/claude_desktop_config.json
Windows: %APPDATA%\\Claude\\claude_desktop_config.json

Add this configuration:

{
  "mcpServers": {
    "meddevices": {
      "command": "mcp-meddevices",
      "args": [
        "--api-key", "YOUR_API_KEY",
        "--endpoint", "https://api.syftbox.net"
      ],
      "env": {
        "SYFTBOX_SERVICE_ID": "med-devices",
        "SYFTBOX_DATASITE": "st-marys-hospital.com"
      }
    }
  }
}
Step 3: Restart Claude Desktop

After saving the configuration, restart Claude Desktop for the changes to take effect.

About the Dataset

This dataset contains comprehensive information about medical devices and equipment used across St. Mary's Hospital network, including installation records, maintenance schedules, and operational status.

Data Sources

  • Hospital equipment management systems
  • Manufacturer installation records
  • Maintenance service logs
  • Equipment inventory audits

Coverage

The dataset covers all major departments including emergency, intensive care, radiology, and surgical units. Imaging equipment and life support systems are particularly well-documented.

Quality Assurance

All records undergo verification by biomedical engineers and cross-referencing with manufacturer databases. Serial numbers and installation dates are validated against official documentation.

API Access

Access PHI-safe medical device data for operational analysis, maintenance planning, and safety monitoring.

Python SDK

import syft_hub as sh

# Initialize client
client = sh.get_client()

# Connect to medical device dataset
dataset = client.get_service(
    datasite="st-marys-hospital.com",
    name="med-device-records"
)

# Search for device alerts
results = dataset.search(
    query="critical ventilator alerts ICU maintenance",
    top_k=15
)

# Query device status
response = dataset.chat(
    message="What MRI scanners need maintenance this week?"
)

HTTP API

curl -X POST "https://api.syftbox.net/v1/services/med-devices/search" \
  -H "Content-Type: application/json" \

    "query": "surgical robot maintenance schedule",
    "top_k": 10,

  }'
