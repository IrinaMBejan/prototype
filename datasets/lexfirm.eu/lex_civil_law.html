The collection spans multiple languages and includes real-time updates from legal databases and court decisions.

About the Dataset

This dataset contains comprehensive European civil law materials including case law, civil codes, legal commentary, and regulatory frameworks from across the European Union and associated jurisdictions.

Data Sources

  • Official court decisions and judgments from EU member states
  • National civil codes and statutory legislation
  • Legal commentary and academic analysis
  • EU regulatory instruments and directives

Coverage

The dataset covers 27 EU member states plus associated jurisdictions, with comprehensive coverage of contract law, property rights, tort liability, and civil procedure. German and French legal systems are particularly well-represented.

Quality Assurance

All legal documents undergo verification by qualified legal professionals and cross-referencing with official legal databases. Citations and references are validated against established legal authorities.

# Load the dataset

# Basic search

# Advanced search with filters
results = legal.search(
    'data protection GDPR',
    top_k=20,

        'type': 'Legal Memo',

Model Context Protocol (MCP)

Installation Options

Claude MCP CLI (Recommended)

Install using the Claude MCP command line interface:

claude mcp add --transport http syftbox_legal https://mcp.syftbox.net/lex-civil

Usage

Once installed, the Lex Civil Law service will be available as an MCP tool in Claude Desktop conversations.

Example usage in Claude:

"Use the legal tool to search for information about..."

Verification

After installation, you should see "Lex Civil Law" listed in your Claude Desktop MCP tools list.

Manual Installation

Step 1: Install the MCP server package
npm install -g @syftbox/mcp-legal
Step 2: Add to Claude Desktop config

Edit your Claude Desktop configuration file:

macOS/Linux: ~/.config/claude/claude_desktop_config.json
Windows: %APPDATA%\\Claude\\claude_desktop_config.json

Add this configuration:

{
  "mcpServers": {
    "legal": {
      "command": "mcp-legal",
      "args": [
        "--api-key", "YOUR_API_KEY",
        "--endpoint", "https://api.syftbox.net"
      ],
      "env": {
        "SYFTBOX_SERVICE_ID": "lex-civil",
        "SYFTBOX_DATASITE": "lexfirm.eu"
      }
    }
  }
}
Step 3: Restart Claude Desktop

After saving the configuration, restart Claude Desktop for the changes to take effect.
